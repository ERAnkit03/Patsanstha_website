## 📖 Overview

This project is a modern frontend website designed for a cooperative financial organization. It provides customers with an intuitive interface to explore financial services, loan schemes, deposit plans, and organization information.

The application follows modern UI/UX principles with reusable React components, responsive layouts, and smooth user interactions.

> **Note:**  
> This repository is a portfolio version of the project. Client-specific branding, confidential information, and proprietary assets have been removed.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/Patsanstha_website.git
```

## Navigate to Project

```bash
cd Patsanstha_website
```

## Install Dependencies

```bash
npm install
```

## Start Development Server

```bash
npm run dev
```

The application will start at:

```
http://localhost:5173
```

---

# 📦 Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
